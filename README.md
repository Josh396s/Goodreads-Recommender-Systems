# Goodreads Recommender Systems
This project implements a multi-task recommender system for book reviews using the Goodreads dataset

## Project Overview
The system addresses two distinct recommendation challenges based on user-book interaction data:
- Read Prediction: Predicting whether a user will read a specific book (Binary Classification)
- Rating Prediction: Predicting the numerical star rating a user would assign to a book (Regression)

## Methodology
1. Read Prediction
To predict user-book interactions, a hybrid strategy was employed to outperform basic popularity baselines:
  - Popularity-Based Filtering: The model identifies a set of "popular books" that account for a specific percentage (threshold) of total interactions in the dataset.
  - Jaccard Similarity: This neighborhood-based approach calculates the similarity between sets of users who interacted with a target book and those who interacted with books the user has previously read.
  - Combined Weighted Logic: Predictions are generated by combining a weighted score from both the popularity and Jaccard similarity modules to determine the final binary output.

2. Rating Prediction
For the regression task, the model predicts star ratings using a latent factor approach:
  - SVD (Singular Value Decomposition): Utilizing the scikit-surprise library, a matrix factorization model was trained to uncover latent features for both users and items.
  - Feature Engineering: The dataset was augmented with features including user and item rating counts, as well as average ratings, to better capture personal and product biases.
  - Normalization: Features were scaled using a MinMaxScaler to improve model convergence and performance.

## Tech Stack
- Language: Python
- Libraries: pandas, numpy, scikit-learn, scikit-surprise
- Environment: Jupyter Notebook / Python Script

## Evaluation Results
The model's performance was measured using standard recommendation metrics:
  - Read Prediction: Evaluated based on Classification Accuracy on a validation set containing 50% positive and 50% negative samples.
  - Rating Prediction: Evaluated using Mean Squared Error (MSE), achieving a validation score of approximately 1.4177.
